## ЁЯЪА How to Use This Tutorial
1. Clone or download the repository:
```bash
   git clone https://github.com/AmiARMiess/html-tutorial.git
   cd html-tutorial
```
2. Open the folder in your code editor.
3. Navigate to a chapter folder (e.g., `01-basics/`).
4. Open any `.html` file in your browser to view the rendered output.
5. Edit the markup, save, and refresh your browser to see changes instantly.

ЁЯТб **Pro Tip:** Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code for automatic browser reloading on save.
