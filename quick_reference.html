📘 HTML Quick Reference Note

A concise, printable cheat sheet for modern HTML5 development. Save as html-notes.html and open in any browser.

📄 Basic Document Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Page description for SEO">
    <title>Document Title</title>
  </head>
  <body>
    <!-- Visible content -->
  </body>
</html>

🔤 Text & Inline Elements

  • <h1>–<h6> – Headings (one <h1> per page)
  • <p> – Paragraph
  • <strong> / <em> – Semantic importance / emphasis
  • <a href="url"> – Hyperlink (use target="_blank" carefully)
  • <code> / <pre> – Inline & block code
  • <br> / <hr> – Line break / thematic divider (void elements)

🖼️ Media & Embeds

<!-- Responsive Image -->
<img src="photo.jpg" alt="Descriptive text" loading="lazy" width="800" height="600">

<!-- Audio -->
<audio controls>
  <source src="podcast.mp3" type="audio/mpeg">
  Your browser does not support audio.
</audio>

<!-- Video -->
<video width="640" height="360" controls poster="thumb.jpg">
  <source src="clip.mp4" type="video/mp4">
</video>

🏗️ Semantic Layout Structure

<header>    <!-- Site/page header or section header -->
<nav>       <!-- Primary navigation -->
<main>      <!-- Unique core content (use once per page) -->
  <article> <!-- Independent, distributable content -->
  <section> <!-- Thematic grouping of content -->
  <aside>   <!-- Tangentially related content -->
</main>
<footer>    <!-- Page or section footer -->

📝 Accessible Forms

<form action="/api/submit" method="POST">
  <label for="name">Full Name</label>
  <input type="text" id="name" name="name" autocomplete="name" required>

  <label for="email">Email Address</label>
  <input type="email" id="email" name="email" placeholder="you@example.com" required>

  <label for="role">Role</label>
  <select id="role" name="role">
    <option value="user">User</option>
    <option value="admin">Admin</option>
  </select>

  <button type="submit">Submit</button>
</form>

Rule: Every form control must have a programmatically associated <label>.

✅ Modern Best Practices

  • Always declare lang on <html>
  • Prefer semantic elements over <div> / <span>
  • Add alt to all meaningful images; use alt="" for decorative ones
  • Use width & height on media to prevent layout shift (CLS)
  • Validate markup regularly: W3C Validator
  • Test with keyboard navigation & screen readers
